# Youth Faith Course Overview

| 课程名称 | 课程主要内容 | 负责老师 |
|---|---|---|
| “开班”致辞 |  | 待定 |
| 青少年分享信仰情况 | 引导大家分享自己对于神的认知，了解真实信仰根基，鼓励大家建立做见证的习惯和勇气。 | 黄姊妹、钱 |
| 信仰 & 科学/理性相关 | 面对生物、地理、历史课和社会宣传中关于世界来源的说法，学习分辨科学观察、理论解释和无神论世界观，建立基督徒世界观。 | 钱、崔 |
| 人性/法律相关 | 讨论人性、善恶、法律、战争、罪、自由意志等问题，建立基督徒人生观和价值观。 | 王爱 |
| 金钱话题 | 讨论金钱、工作、财富、十一奉献，建立基督徒对金钱与财富的观念。 | 小鱼 |
| 基督徒在高中/大学生活 | 过来人的经历、建议和提醒，包括团契、学业、灵修、交友、恋爱、危险求助、信仰身份、网络文化等。 | 玉丽 |
| 信仰 & 艺术/感性相关 | JD信仰与艺术、摄影、美术、音乐等话题。 | 朱 |

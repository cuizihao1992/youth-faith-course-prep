# 信仰 & 科学/理性相关

## 最终讲课文件

### 01_content_outline.md

内容提纲。

用来把握这堂课的主题、结构、重点、经文和例子。

### 02_session_script.md

课程脚本。

用来实际讲课，包含开场、转场、互动问题、活动和结束祷告。

## 使用建议

讲课前主要看两个文件：

1. `01_content_outline.md`
2. `02_session_script.md`

如果需要调整开场、结尾或中间逻辑，再看 `04_opening_options.md`、`05_closing_options.md`、`06_middle_flow.md`。

## 这堂课不追求

- 不追求讲完所有科学问题。
- 不追求把无神论全部反驳完。
- 不追求让孩子马上想通一切。

## 这堂课追求

让孩子离开父母、进入高中或大学、将来有一天动摇时，还记得：

**我可以把问题带到神面前。**

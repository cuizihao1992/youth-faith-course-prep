# Youth Faith Course Outline

| 课程名称 | 课程主要内容 | 负责老师 |
|---|---|---|
| “开班”致辞 |  | 待定 |
| 青少年分享信仰情况 | 切入点：<br>1、大家都是信二代、信三代，你父母的神是否成为了你的神，你有经历过神的同在/声音/感动吗？<br>2、多次祷告不应允，即便应允，分不清是否合还是自身努力；或者现在可以依靠的东西太多，想自己解决，不想祷告怎么办。<br><br>引导大家分享自己对于神的认知、了解大家的真实信仰根基、鼓励大家建立做见证的习惯和勇气。 | 黄姊妹、钱 |
| 信仰 & 科学/理性相关 | 切入点：生物/地理/历史课/社会宣传的世界来源、科学宣传的世界来源。与圣经的教导相冲突，如何看待这种冲突。考试时如果遇到这种题目要怎么回答。等等。<br><br>核心信仰内容：建立基督徒的世界观 | 钱、崔 |
| 人人性/法律相关 | 切入点：<br>1、性善论和性恶论。为什么社会会有道德、法律、教育、医疗、宗教，这些是带领人向善吗？<br>2、为什么会有战争？神为什么允许有战争？为什么不灭绝恶人，只留下好人？为什么要吩咐不让吃善恶树的果子，不吩咐不就没有罪了吗？<br><br>核心信仰内容：基督徒的人生观/价值观。神按自己的形象样式造人，什么是人，人应该是怎么样的；堕落以后，人是怎么样的，人的罪性、罪行等。 | 王爱 |
| 金钱话题 | 切入点：什么是金钱、为什么很多人都拼命得去挣钱。读书/将来工作的目的就是为了挣钱吗，挣多少钱才算够，等等<br>十一奉献<br>核心内容：基督徒对金钱与财富的观念 | 小鱼 |
| 基督徒在高中/大学生活 | 过来人的一些经历、建议、忠告<br>高中/大学生活的几个重要提醒（大学团契、学业和信仰、灵修建议、交友、谈恋爱、婚前性行为）等<br>遇到危险、困难，如何求助<br>当被问到是否有信仰（比如入团），是什么信仰？是否可以入党，该如何回答？被老师要求签无信仰保证书怎么办？<br>追星、穿着打扮、网络热梗、同性恋等话题 | 玉丽 |
| 信仰 & 艺术/感性相关 | JD信仰与艺术、摄影、美术、音乐等等话题 | 朱 |
